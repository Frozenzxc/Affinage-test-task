# AFFINAGE test task

## Подготовка к запуску проекта

Необходимо выполнить установку необходимых пакетов с помощью команды:
```
npm i
```

## Запуск проекта

После ввода следующей команды в консоль, будет собран Gulp'ом и запущен проект по адесу http://localhost:3000/ (автозапуск браузера отключен)
```
npm start
```

## Дополнительные команды

Сборка проекта в папку docs

```
npm run build
```
